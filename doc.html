Overcooked Gourdy's Directory Library Documentation
Never forget to add a comma to your ending brackets, it may not work! (Except if the thing is the last one)

1. Text Page


"My Text Page": {
    type: "text",
    content: `Normal text

Supports multiple lines

You can also encode:
BASE64:SGVsbG8=
REV:olleH`
}
Supports "BASE64:___==" encoded text, and "REV:___" reversed text so the ai po po dont catch your doings alongside folder names, image names, dividers, etc. Dont worry, itll cliently decode!

2. Image Page


"My Image Page": {
    type: "slideshow",
    slides: [
        { image: "https://picsum.photos/800/500" }
    ]
}

3. Text Slideshow


"My Text Slideshow": {
    type: "slideshow",
    slides: [
        { text: "Slide 1" },
        { text: "Slide 2" }
    ]
}

4. Image Slideshow


"My Image Slideshow": {
    type: "slideshow",
    slides: [
        { image: "img1.png" },
        { image: "img2.png" }
    ]
}

5. Mixed Slideshow


"My Mixed Slideshow": {
    type: "slideshow",
    slides: [
        { text: "Hello", image: "img.png" },
        { text: "Only text" }
    ]
}

6. Folder


"My Folder": {
    type: "folder",
    items: {
        "Page": {
            type: "text",
            content: "Hello world"
        }
    }
}

7. Nested Folder


"Main Folder": {
    type: "folder",
    items: {
        "Sub Folder": {
            type: "folder",
            items: {
                "Deep Page": {
                    type: "text",
                    content: "Deep content here"
                }
            }
        }
    }
}

8. Dividers


"── MAIN SECTION ──": {
    type: "divider"
}
Used to visually separate sections inside folders.
